## 使用流程

### 1. 创建虚拟环境
终端运行
`python -m venv venv_name`
### 2. 激活虚拟环境
等待虚拟环境创建完成后激活虚拟环境
`venv_name\Script\activate`
### 3. 安装依赖
`pip install -r requirements.txt`
### 4. 设置本地打印机名称
* 打开打印机和扫描仪设置界面，找到要使用的打印机，
![alt text](image-1.png)
* 修改打印机名称为“辅助打证”
![alt text](image.png)
### 5. 运行代码
终端运行
`python app.py`
